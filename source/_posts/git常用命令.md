## 安装
Linux（本人用的是Ubuntu）下的安装十分方便，进入命令行执行安装命令即可：
```bash
sudo apt-get install git
```
Windows的童鞋可以自行到 [官网](https://git-scm.com/downloads) 下载完成安装
